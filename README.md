# Forge Plugin Manager

Desktop application for managing [Forge](https://reumbra.dev/forge) plugins in Claude Cowork.

## Features

- License activation and management
- Plugin catalog with one-click install
- Automatic update detection
- Cowork directory auto-detection (Windows, macOS, Linux)
- Feedback submission

## Download

Download the latest release for your platform from [Releases](https://github.com/maselious/forge-plugin-manager/releases).

| Platform | File |
|----------|------|
| Windows | `.msi` or `.exe` |
| macOS (Apple Silicon) | `.dmg` |
| macOS (Intel) | `.dmg` |
| Linux | `.AppImage` or `.deb` |

## Development

```bash
pnpm install
pnpm tauri dev
```

Requires [Rust](https://rustup.rs/) and platform-specific dependencies. See [Tauri prerequisites](https://v2.tauri.app/start/prerequisites/).
